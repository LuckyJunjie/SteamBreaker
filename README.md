## 游戏概述

**Steam Breaker** 是一款类《重装机兵》玩法的蒸汽朋克航海RPG。玩家驾驶自己DIY的蒸汽船，与个性伙伴一起讨伐海洋巨兽与海盗，在无尽的群岛间书写属于自己的退隐传说。

### 核心特色

- 🚢 **船只DIY系统** — 船体/锅炉/操舵室/炮台/特殊装置自由改装
- ⚔️ **回合制战斗** — 射程环系统带来策略深度
- 👥 **伙伴羁绊** — 5位可招募伙伴，好感度触发专属剧情
- 🏴‍☠️ **赏金系统** — 讨伐海兽、海盗、帝国军官
- 🗺️ **开放海域** — 多结局、多支线、多小游戏

## 快速开始

### 环境要求

- Godot 4.x
- Git

### 克隆项目

```bash
git clone https://github.com/LuckyJunjie/SteamBreaker.git
cd SteamBreaker
```

### 运行游戏

```bash
# 使用 Godot 打开项目
godot --path . &
```
