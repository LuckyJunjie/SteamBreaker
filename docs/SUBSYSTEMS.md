## 1. 船只DIY系统（核心）

### 1.1 部件槽位与属性影响

| 部件类型 | 基础属性 | 可衍生效果 | 改装示例 |
|---------|----------|-----------|----------|
| **船体** | 耐久、载重、炮位数量、基础防御 | 特殊能力（潜航、破冰、隐形） | 侦察艇壳（高闪避）、铁甲舰壳（高耐久）、潜水壳（可下潜） |
| **锅炉** | 航速、过热阈值、机动值回复 | 过热惩罚、特殊燃料效果 | 单胀机（平衡）、复式膨胀机（高航速）、雷钢锅炉（高过热阈值但易爆） |
| **操舵室** | 转向、可装备特技数量、迎击加成 | 自动导航（回避陷阱）、战术指令 | 手动舵（低成本）、陀螺仪（高转向）、AI参谋（额外战术指令） |
| **主炮** | 伤害、射程、装填时间、过热消耗 | 弹种切换、范围攻击 | 卡隆炮（高伤低命中）、鱼雷管（无视护甲）、高压水炮（击退） |
| **副炮** | 伤害、拦截能力、射速 | 自动迎击、多目标 | 转管炮（高拦截）、深弹发射器（反潜）、信号弹（照明） |
| **特殊装置** | 技能效果、冷却回合 | 团队辅助、环境互动 | 撞角（近战冲撞）、烟幕（闪避提升）、修理臂（战斗中修复）、声呐（探隐形） |

### 1.2 改装限制与平衡

- **重量限制**：总重 ≤ 船体载重，超出后航速和转向大幅下降，甚至无法出航。
- **炮位限制**：每个炮位只能安装一门主炮/副炮，船体决定炮位数量（如小型船体2主1副，大型4主2副）。
- **过热机制**：每个武器和锅炉都有"热值"。攻击或加速会累积**过热值**，超过阈值进入过热状态，若干回合无法使用攻击/加速。
- **专属配件**：部分伙伴或剧情解锁专属部件（如珂尔莉的"鹰眼瞄准器"）。

### 1.3 改装界面设计

- **画布**：左侧2D或简易3D船体模型，右部件列表。拖动部件到槽位。
- **实时反馈**：下方显示重量/载重进度条、航速/耐久变化。
- **预览开关**：支持"试航"模拟战斗（不消耗实际资源）。

### 1.4 获取方式

- 购买（港口船坞）
- 打捞（沉船探险）
- 制作（图纸+废料）
- 赏金奖励
- 伙伴羁绊任务赠送

## 2. 动态多范围战斗系统

### 2.1 射程环机制数值

| 射程 | 移动消耗机动值 | 命中加成（攻方） | 闪避加成（守方） | 可用武器限制 |
|------|--------------|----------------|----------------|--------------|
| 远 | 高（约40%） | -20% | +30% | 主炮、鱼雷（需目标在本半场）、SE（部分） |
| 中 | 中（20%） | 0% | 0% | 全部 |
| 近 | 低（10%） | +30%近战武器 | -15% | 副炮+50%伤害，主炮命中-20%，撞角/接舷可用 |

- **机动值**：基础值10点，由锅炉决定每回合回复量（如单胀机回复3，雷钢回复5但过热快）。
- **移动消耗**：从当前环到目标环的消耗 = 环数差×基础消耗。
- **强制位移**：部分技能可将敌方推远或拉近。

### 2.2 部位攻击命中率与效果

玩家可"瞄准"敌方特定部位，命中率降低，但收益明显。

| 部位 | 基础命中率 | 效果 |
|------|-----------|------|
| 船体 | 100% | 削减耐久 |
| 锅炉 | 70% | 过热强制+30，速度降低 |
| 操舵室 | 60% | 下一回合无法移动，转向归零 |
| 炮位（指定武器） | 65% | 摧毁该武器（战斗中不可用） |
| 特殊装置 | 55% | 禁用SE技能X回合 |

- 部分技能可提高部位命中率（如珂尔莉"精确射击"+20%）。
- 命中部位后播放损坏特效（冒烟、爆炸）。

### 2.3 战斗UI流程

1. **回合开始**：显示当前射程环、机动值、双方状态。
2. **移动选择**：点击环图标（远/中/近）或拖动船标，显示移动后效果。
3. **指令面板**：武器列表（显示过热状态、弹药）、特殊装置、修理、道具、防御。
4. **部位选择（可选）**：开启瞄准模式时出现部位轮盘。
5. **攻击动画**：炮弹/鱼雷飞行轨迹，迎击判定。
6. **损伤结算**：跳字，部位特效。
7. **回合结束**：轮到敌方。

### 2.4 迎击系统

- 副炮自动拦截敌方炮弹/导弹，拦截成功率 = 副炮拦截值 + 操舵室加成。
- 每次迎击消耗副炮"专注值"（每个副炮独立，每回合回复1点）。
- 可命令副炮"全力迎击"（消耗所有专注值，提高拦截率但无法攻击）。

### 2.5 船只状态效果

| 状态 | 效果 | 解除方式 |
|------|------|----------|
| 过热 | 无法使用武器，移动消耗×2 | 等待2回合或使用道具"海水注入" |
| 瘫痪 | 无法行动（操舵室被毁） | 修理 |
| 漏水 | 每回合失去5%耐久 | 修理装置/道具"堵漏毯" |
| 火灾 | 每回合失去10%耐久，锅炉过热加速 | 灭火装置/道具 |
| 隐形 | 无法被瞄准，无部位命中 | 烟幕/特殊装置，持续2回合 |

## 3. 伙伴与羁绊系统

### 3.1 伙伴数据结构

```gdscript
# 伙伴基础数据 (Resource)
- id: String
- name: String
- species: String
- portrait: Texture
- base_stats: Dictionary (hp,攻击,防御等)
- skills: Array[Skill]
- likes: Array[ItemID]      # 礼物喜好
- dislike: Array[ItemID]
- affection: int (0-100)
- story_unlocked: bool
```

### 3.2 好感度获取途径

- 赠送喜好礼物：+5~+15
- 对话正确选项：+2~+8
- 共同战斗胜利：+1（小型战斗），+3（赏金战）
- 完成羁绊支线：+20
- 赠送厌恶物品：-5
- 战斗中让伙伴濒死：-10

### 3.3 羁绊等级与奖励

| 好感度 | 等级 | 解锁奖励 |
|--------|------|----------|
| 0-20 | 陌生 | 基础对话 |
| 21-40 | 相识 | 个人背景故事解锁 + 一个基础技能 |
| 41-60 | 信任 | 羁绊支线第一章 + 专属船只部件蓝图 |
| 61-80 | 亲密 | 羁绊支线第二章 + 高级技能 + 战斗对话彩蛋 |
| 81-100 | 灵魂 | 羁绊结局 + 最终技能 + 可继承至新游戏+ |

### 3.4 羁绊支线设计（以珂尔莉为例）

- **第一章 - 失落的飞行舰队**：在某岛发现坠毁的鸟族飞艇残骸，珂尔莉要求调查，发现是帝国击落。战斗后得到"鹰眼瞄准器"（部件）。
- **第二章 - 天空墓场**：前往高空浮岛，与叛徒鸟族队长战斗，珂尔莉觉醒技能"风暴之眼"（全队命中+30%一回合）。
- **结局**：珂尔莉决定建立新的飞行队，邀请玩家担任名誉队长。若好感100且选择与她隐退，结局画面为两人在云中岛屿经营飞行驿站。

### 3.5 战斗中的伙伴技能

- 每个伙伴有最多3个主动技能，随羁绊等级解锁。
- 技能消耗"士气"值（每场战斗独立，击杀/受伤回复）。
- 示例技能：
  - 珂尔莉："狙击桅杆"（高几率破坏敌方操舵室）
  - 铁砧："紧急抢修"（消耗士气，恢复船体耐久）
  - 深蓝："洋流牵引"（将敌方拉近一个射程环）

## 4. 赏金系统

### 4.1 悬赏数据结构

```gdscript
- bounty_id: string
- name: string
- rank: 海域级/史诗级
- reward_gold: int
- reward_items: Array[Item]
- spawn_location: string or "random"
- encounter_condition: string (天气、时间、剧情进度)
- dialogue: 战斗前/中/后的特殊台词
- special_mechanics: 战斗机制（如"每回合召唤小怪"）
```

### 4.2 悬赏追踪方式

- **公告板**：港口悬赏公会可接取，提供大致海域。
- **情报收集**：酒馆NPC对话、漂流瓶、地图残片可精确位置。
- **随机遭遇**：在海域漫游时，低概率直接遇上未接取的高阶赏金首（惊喜/惊吓）。

### 4.3 赏金战斗机制示例

| 赏金首 | 特性 | 策略 |
|--------|------|------|
| 铁牙独眼鲨 | 高攻、会撞船 | 保持中远距离，用鱼雷；打掉其铁牙可掉材料 |
| 幽灵船悔恨女王 | 吸收生命、免疫物理 | 使用圣水弹（道具），或攻击桅杆上的幽灵灯 |
| 帝国叛将黑炉 | 高甲、多炮台 | 优先破坏主炮，再用穿甲弹攻击锅炉 |

### 4.4 赏金排行榜与重复挑战

- 每次击败赏金首，公会记录"最小损伤"、"最快回合"等成就。
- 可重复挑战已击败赏金首（但奖励减半，主要用于刷材料）。

## 5. 经济与道具系统

### 5.1 货币体系

- **金克朗**：通用，购买普通物品，赏金主要奖励。
- **帝国债券**：稀有，完成帝国任务、探明遗迹获得，用于购买稀有图纸、特殊伙伴。
- **废料**：分解船只部件获得，用于制作/改装。

### 5.2 道具分类与使用场景

| 类别 | 示例 | 使用时机 |
|------|------|----------|
| 修理 | 木板包（恢复300耐久）、铁皮装甲（提升防御3回合） | 战斗内/外 |
| 弹药 | 穿甲弹、燃烧弹、圣水弹 | 战斗内装填（部分主炮消耗弹药类型） |
| 补给 | 锅炉清洁剂（降低过热值）、烟幕桶 | 战斗内 |
| 礼物 | 陈年朗姆、机械油、发光海藻 | 港口对话赠送 |
| 探险 | 声呐屏（显示周边敌舰）、拖网（捞材料） | 非战斗地图使用 |

### 5.3 交易与物价

- 每个港口有不同物价（如工业港机械零件便宜，渔业港食物便宜）。
- 帝国债券商人只在特定港口出现。

## 6. 小游戏系统设计

### 6.1 锅炉骰子（酒馆）

- **玩法**：掷三颗六面骰，每面为锅炉压力值（1-6）。组合得分：
  - 三个相同：10倍点数
  - 顺子（1-2-3 至 4-5-6）：20金
  - 对子+另一个：点数之和
- 可"再次掷出"两次，每次下注10金起。

### 6.2 海鸟赛跑

- 选择一只海鸟（属性：速度、耐力、运气）。
- 赛道有障碍（浪、海风），玩家可消耗"哨子"让鸟短暂加速。
- 押注倍率根据鸟的赔率变化。

### 6.3 炮术射击

- 鱼眼视角，移动靶出现，限时30秒。
- 命中环数得分，总分兑换炮弹或副炮图纸。

### 6.4 齿轮拼图（机械师公会）

- 盘面上有多个齿轮，旋转其中一些使所有齿轮联动。
- 步数限制，解谜成功获得"古代零件"（兑换部件蓝图碎片）。

## 7. 隐退与结局系统

### 7.1 触发方式

- 任何时候回到**铁锈湾**，与老渔夫对话"我想隐退"。
- 强制存档，并提醒"这是终点"。

### 7.2 结局判定条件

根据以下权重计算最终结局：

| 条件 | 权重 |
|------|------|
| 羁绊最高伙伴（好感＞80） | 伙伴结局 |
| 金币 > 100000 | 富豪结局 |
| 击败所有史诗赏金首 | 传说结局 |
| 主线最终选择（重启/摧毁/保留蒸汽机） | 影响世界观 |
| 无伙伴好感>50且金币<50000 | 普通隐退 |
| 伙伴全部死亡/离队且主线失败 | 悲剧结局 |

### 7.3 结局呈现方式

- 幻灯片 + 旁白，展示玩家与伙伴/世界的后续。
- 例如伙伴结局：珂尔莉与玩家在云端驿站，铁砧在岸边开修理铺，深蓝与鲸群巡游...
- 结局后播放制作人员名单，并提供"新游戏+"选项（继承部分道具/图纸）。

## 9. UI/UX要点

- **船只改装**：直接拖拽部件，重量超限时红框警告。
- **战斗界面**：射程环采用环形指示器，点击环即可移动；武器按钮过热时禁用并显示冷却倒计时。
- **港口场景**：可交互点（船坞、酒馆、公会、商店）高亮，支持快速旅行。
- **地图**：手绘风格，标注赏金位置、天气信息。
- **伙伴状态**：在航行界面右下角显示伙伴头像+情绪图标（基于好感度）。

## 10. 技术实现提示（Godot 4）

- **数据驱动**：所有部件、道具、赏金、伙伴定义为`Resource`，便于扩展和本地化。
- **战斗状态机**：使用`StateMachine`节点，状态：`Idle` -> `PlayerMove` -> `PlayerAction` -> `EnemyTurn` -> `CheckEnd`。
- **射程环**：通过`CanvasItem`绘制环形图标，移动动画使用`Tween`。
- **部位损伤**：船体模型挂载多个`Area3D`，定义`hitbox`分组，炮弹hit检测时判断碰撞的节点名。
- **存档**：使用`ResourceSaver`保存玩家数据（`PlayerData`资源），包括船只部件数组、伙伴好感度数组、赏金进度。
- **性能优化**：海洋场景使用`WorldEnvironment` + 低多边形海面，远处LOD。
