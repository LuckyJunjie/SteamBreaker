# Steam Breaker（蒸汽破浪号）游戏设计文档

---

## 一、游戏概述

| 项目 | 内容 |
|------|------|
| **游戏名称** | Steam Breaker（蒸汽破浪号） |
| **类型** | 回合制航海RPG + 船只自由改装 + 开放海域探索 |
| **平台** | PC（Steam），后续可移植主机 |
| **目标玩家** | 喜欢《重装机兵》、沙盒探索、机械改装与伙伴剧情的玩家 |
| **核心体验** | 驾驶自己DIY的蒸汽船，与个性伙伴一起讨伐海洋巨兽与海盗，在无尽的群岛间书写属于自己的退隐传说 |

**一句话卖点**：*“开着你的铁船，带着你的怪人，去把悬赏令上的家伙轰成碎片——然后回酒馆吹一整个雨季的牛。”*

## 二、世界观与故事背景

### 时代
“大蒸气纪元”第187年。一场被称为“大沸腾”的灾难使海平面上升，大陆碎裂成数以万计的群岛。幸存者在蒸汽机与旧世界残骸之上建立了新的文明——**雾之帝国**。

### 核心冲突
帝国依靠垄断“雷钢”（一种高能金属）维持统治，而广袤的外环海域则成为法外之地。赏金猎人、海盗、遗迹猎人、疯狂科学家在此角逐。玩家扮演一名失忆的船长，从一艘破旧的二手船“蒸汽破浪号”开始，逐步揭开“大沸腾”的真相。

### 世界观特色
- **蒸汽朋克 + 航海**：船即是家、武器、伙伴。海水被染成铁锈色，天空永远有薄雾，灯塔用齿轮驱动。
- **非人类种族**：鱼人、海鸟族、珊瑚精、机械改造人、鲸语者等。
- **活着的海洋**：有智慧的海流、巨大的浮游生物城市、沉没的帝国遗迹。

## 四、伙伴系统（羁绊为核心）

### 可招募伙伴（示例）
| 名字 | 种族 | 职业 | 个性 | 羁绊剧情线 |
|------|------|------|------|------------|
| 珂尔莉 | 鸟族 | 瞭望手/狙击手 | 傲娇但怕雷声 | 寻找失踪的飞行舰队 |
| 铁砧 | 机械改造人 | 轮机长/修理师 | 沉默寡言，嗜喝机油 | 找回被洗掉的记忆 |
| 深蓝 | 鱼人 | 导航员/潜航士 | 内向，能与鱼群对话 | 阻止帝国污染其故乡海沟 |
| 贝索船长 | 人类（老人） | 炮术长/老兵 | 酒鬼，爱讲旧事 | 与昔日叛徒决斗 |
| 磷火 | 珊瑚精（幼体） | 医疗/炼金 | 天真但拥有毒素 | 寻找“生命之核”进化 |

### 羁绊机制
- 每个伙伴有独立的**好感度**（0-100），通过赠送礼物、对话选择、共同战斗提升。
- 好感度达到阈值解锁**羁绊剧情**（专属任务），完成后获得该伙伴的**特殊技能**或**船只改造蓝图**。
- 好感度达到100时，触发**羁绊结局**（例如与伙伴一同隐退、继承其遗志等）。

## 五、船只DIY系统（核心玩法的深度）

### 船体基础属性
- **耐久**（HP）
- **载重**（影响可装备总重量）
- **航速**（影响战斗中先手与回避率）
- **转向**（影响战斗中移动效率）

### 可改装部件（五大类）
| 部件槽位 | 功能 | 改装举例 |
|----------|------|----------|
| **船体**（底盘） | 决定基础耐久、载重、可安装炮位数量 | 轻型侦察船壳 / 铁甲战列舰壳 / 潜水艇壳（可下潜） |
| **锅炉**（引擎） | 决定航速、输出功率（影响过热阈值） | 单胀式蒸汽机 / 复式膨胀机 / 雷钢超临界锅炉 |
| **操舵室**（C装置） | 决定转向、可安装特技数量 | 手动操舵轮 / 自动陀螺仪 / 战术参谋AI |
| **主炮** | 高伤害、消耗大 | 24磅卡隆炮 / 旋转式鱼雷管 / 高压水炮 |
| **副炮** | 低伤害、可拦截敌方弹药 | 速射转管炮 / 火箭深弹发射器 |
| **特殊装置**（SE） | 辅助/战术技能 | 蒸汽撞角 / 烟幕发生器 / 修理用机械臂 / 声呐浮标 |

### 改装限制（保持策略性）
- 每个部件有**重量**，总重不能超过载重。
- **过热条**：连续使用高能耗武器或极限加速会使锅炉过热，过热后若干回合内无法行动。
- **空间限制**：小船体只能装2门主炮，大船体可装4门。

### 外观可视化
- 换上不同船体模型变化明显；加装撞角后船头出现尖刺；更换锅炉后烟囱数量和形状改变。

## 六、战斗系统（动态多范围战斗）

### 核心机制：射程环（Distance Ring）
战斗场景从传统的“左右对峙”改为**环形射程**概念。玩家船可在三个射程环之间**每回合消耗机动值自由移动**：

| 射程 | 效果 | 可用武器 | 特殊规则 |
|------|------|----------|----------|
| **远距离**（30-50链） | 先手优势，敌方命中下降30% | 主炮（鱼雷无效），副炮无法使用 | 可执行“撤退” |
| **中距离**（10-30链） | 标准交战区 | 所有武器正常 | 可尝试“冲角准备” |
| **近距离**（0-10链） | 高伤害、高风险 | 副炮伤害+50%，主炮命中-20%，可使用撞角/接舷战 | 可执行“登船白刃战” |

### 战斗流程（回合制，但有实时机动决策）
1. **机动阶段**：选择移动目标射程环（消耗机动值，机动值由锅炉功率决定）。
2. **指令阶段**：选择武器攻击、使用特殊装置、修理、使用道具。
3. **迎击阶段**：敌方回合，副炮可自动拦截部分来袭弹药（拦截成功率由操舵室性能与炮手技能决定）。
4. **损伤阶段**：显示船体各模块受损情况（如“锅炉损坏：航速减半”）。

### 部位损伤系统
战斗中可**指定攻击敌方船只的特定部位**（命中率降低但效果显著）：
- **攻击锅炉**：使其过热或减速。
- **攻击操舵室**：使其无法移动或转向。
- **攻击炮位**：暂时禁用该武器。
- **攻击船体**：单纯削减耐久。

### 战术技能（由伙伴或特殊装置提供）
- **珂尔莉技能“桅顶狙击”**：在远距离可精准破坏敌方操舵室。
- **铁砧技能“紧急抢修”**：消耗一回合恢复部分船体耐久。
- **特殊装置“烟幕”**：大幅度提升闪避，持续两回合。

## 七、赏金系统（核心驱动力）

### 悬赏海报
- 每个港口的“赏金公会”贴有通缉令，分为**海域级**（强力单目标）和**史诗级**（剧情BOSS）。
- 部分悬赏目标不是直接战斗，而是需要**追踪与解密**（如跟随漂流瓶线索发现海兽巢穴）。

### 赏金举例
| 名称 | 悬赏金 | 特征 | 掉落 |
|------|--------|------|------|
| “铁牙”独眼鲨 | 3000金 | 巨大鲨鱼，装备铁制下颚，可咬碎船板 | 鲨鱼牙刺（副炮材料） |
| 幽灵船“悔恨女王” | 8000金 | 无人船，会吸收船员生命力 | 亡灵罗盘（特殊装置） |
| 帝国叛将“黑炉” | 20000金 | 驾驶雷钢战列舰的旧帝国将军 | 雷钢核心（最强锅炉材料） |

### 悬赏的随机性
- 部分海域有**随机遭遇的悬赏目标**（类似《重装机兵》的“游走赏金首”），需要在海上漂泊触发遭遇。

## 九、经济与道具系统

### 货币
- **金克朗（Gold Crown）**：普通货币，用于购买普通部件、补给。
- **帝国债券**：声望货币，用于购买稀有图纸、特殊伙伴。

### 道具分类
- **修理类**：木板、铁皮、船用急救包。
- **战斗辅助类**：穿甲弹、高爆弹、烟雾弹、信号弹。
- **探险类**：声呐屏、海上地图、捕鲸叉。
- **伙伴礼物类**：机油（铁砧喜好）、发光海藻（深蓝喜好）、陈年朗姆（贝索喜好）。

## 十、隐退与多结局系统

### 隐退触发方式
- 任何时候回到**初始港“铁锈湾”**，与码头的老渔夫对话，选择“我不想干了”。
- 系统弹出警告：**“你的冒险将会结束，真的上岸吗？”**

### 结局多样性（基于玩家行为）
- **普通隐退**：卖掉船只，在故乡开一家修船铺，偶尔给孩子们讲冒险故事。
- **伙伴结局**：好感度最高的伙伴会与玩家一同隐退（例如与珂尔莉开空中驿站，与深蓝共建海底农场）。
- **富豪结局**：拥有10万金币以上隐退，买下一座小岛成为贵族。
- **传说结局**：击败所有史诗赏金首后隐退，多年后传说你成为“雾之海的海神”。
- **悲剧结局**：重大剧情选择失误导致所有伙伴离去，独自隐退，画面灰暗。

## 十一、主线剧情梗概（无剧透版）

1. **序章**：在铁锈湾被老渔夫救起，获得烂船“蒸汽破浪号”，开始接小赏金维生。
2. **帝国来人**：神秘帝国女军官出现，委托玩家调查“大沸腾”遗迹。
3. **三个势力角逐**：帝国、海盗同盟、深海教父（崇拜海洋极端组织）。
4. **真相**：大沸腾是古代文明留下的“全球蒸汽机”失控导致，玩家必须选择**重启并控制它**、**毁掉它**或**让它继续运转**。
5. **终局**：根据选择，面对不同的最终BOSS（帝国终极战舰 / 远古海神 / 机械神降）。
6. **尾声**：战后世界的变化 + 隐退分支。

## 十二、美术与音乐风格

### 美术方向
- **基调**：雾灰、锈红、铜绿为主色，辅以少量荧光蓝（雷钢光辉）。
- **船只设计**：参考“维多利亚式战列舰 + 裸露的蒸汽管道 + 铆钉结构”。
- **角色设计**：日式动画风格，但服饰融入航海与工业元素（如珂尔莉的披风上有齿轮纽扣）。

### 音乐风格
- **通常海域**：手风琴 + 大提琴 + 轻微工业噪音，营造“海风与蒸汽”的混合感。
- **战斗音乐**：重金属 + 蒸汽汽笛声，快节奏。
- **酒馆音乐**：爱尔兰民谣风格，有吧台音效和杯子碰撞声。

## 十三、技术要点（基于Godot）

- **核心架构**：使用Godot 4的**场景树**与**资源系统**管理船只部件数据（自定义Resource）。
- **战斗系统**：状态机（`StateMachine`）管理"机动 → 指令 → 迎击 → 结算"循环。
- **射程环逻辑**：使用`int distance_ring`变量，并在UI上显示环形指示器。
- **部位损伤**：在船体场景中挂载不同部位的`Area3D`（或2D的`CollisionShape2D`），受击时判断命中的是哪个部位。
- **存档系统**：使用`ConfigFile`或`JSON`保存玩家船只改装、伙伴好感、赏金进度。
- **开发展望**：可先用2D像素风格快速搭建原型（降低开发难度），美术稳定后再升3D。
